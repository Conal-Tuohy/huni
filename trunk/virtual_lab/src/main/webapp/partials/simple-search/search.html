The Humanities Networked Infrastructure (HuNI) is a national ‘virtual laboratory’ project that is being developed as part of the Australian government’s NeCTAR (National e-Research Collaboration Tools and Resources) program. HuNI is using the Linked Data framework to combine information from 28 of Australia’s most significant cultural datasets. These datasets comprise more than 2 million authoritative records relating to the people, objects and events that make up the country’s rich heritage.
